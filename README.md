# Clever-post

Application that manages the daily friend's posts. You can edite or delete them. 

## How to start app:

`npm install` or `yarn`

And then: 

`npm run dev` 

# Built with

<li>Vite</li>
<li>Typescript</li>
<li>React</li>
<li>React-Router</li>
<li>React-redux</li>
<li>Redux-toolkit</li>
<li>SCSS</li>
